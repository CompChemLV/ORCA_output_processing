This repository contains tools that helps with data extraction of ORCA's output. A collection of Python scripts designed for data extraction, and automated processing of output files generated by the ORCA quantum chemistry program.

## Key Features

* **Automated Extraction:** Parsing of ORCA `.out`, `.property.json` and `.inp` files to tabulate key properties such as energies, frequencies, partial charges, and thermochemical parameters.
* **Structure Description:** Extracts internal coordinates from ORCA outputs to automatically construct a **Z-matrix** exported in `.csv` format for easy structural analysis.
* **Data Tidy:** Leverages `pandas` to structure, clean, and organize raw output data into tidy formats (CSV, Excel), ensuring compatibility with modern statistical analysis and visualization pipelines.


## Requirements and Environment

These scripts are designed to be flexible and run on both Linux environments (such as Debian) and Windows systems via distributions like Anaconda or Miniconda.

**Main Dependencies:**
* Python 3.8 or higher
* `pandas`
